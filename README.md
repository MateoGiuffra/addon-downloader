A simple tool to download multiple GitHub repositories at once.

## How to Use

1. **Download the executable**: Get `AddonDownloader.exe` and place it wherever you want your addons folder.

2. **Run the program**: Double-click `AddonDownloader.exe`

3. **Paste your URLs**: Copy and paste text containing GitHub repository URLs. The program automatically detects URLs in any format.

4. **Press Enter twice** to start downloading

5. **Done!**: All repositories will be downloaded into an `AddOns` folder in the same location as the executable.

## Example

If you have a list like this:
```
https://github.com/user1/addon1
https://github.com/user2/addon2.git
Check out this cool addon: https://github.com/user3/addon3
```

Just paste it into the program, press Enter twice, and all three repositories will be downloaded automatically.

## Requirements

- **Git must be installed** on your system
- Windows operating system
- Active internet connection

## Supported URL Formats

El script acepta URLs en estos formatos:

```
C:/Games/TurtleWoW/Interface/AddOns TrainerSkills *https://github.com/refaim/TrainerSkills.git master
```
https://github.com/user1/repo1
https://github.com/user2/repo2.git
TrainerSkills https://github.com/refaim/TrainerSkills.git
```

The program extracts URLs from any text format.
